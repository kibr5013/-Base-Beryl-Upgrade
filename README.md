# -Base-Beryl-Upgrade
Base Beryl testnet is now live — introducing B20 native tokens, faster withdrawals, and major scalability improvements.






📌 Overview

Beryl is Base's second major network upgrade after Azul.

The upgrade focuses on:

Native token infrastructure
Better capital efficiency
Improved scalability
Lower node resource usage

Beryl also demonstrates faster development cycles for Base, launching only weeks after Azul.

✨ Features
🪙 B20 — Native Token Standard

B20 introduces a chain-level token standard designed for:

Stablecoins
Real-world assets (RWAs)
Onchain-native assets

✅ ERC-20 compatible
✅ ERC-2612 permits
✅ Role-based access control
✅ Mint & burn support
✅ Supply caps
✅ Transfer policies
✅ Freeze/seize support
✅ Transfer memos
